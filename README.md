# Banco
## Criação da Classe Banco (Atividade curso Java - Módulo 01)

O projeto está incompleto e com alguns bugs. A ideia era ter a `Interface CaixaEletronico` para que o cliente pudesse chamar os métodos abstratos `cadastrar`, `logar` e `acessarConta`. Fiquei confuso em como implementar o cadastro do cliente. Será que era necessário criar objetos do tipo ContaCorrente e ContaPoupanca para que os métodos `transferencia` e `depositar` funcionassem de forma correta? Além disso faltou implementar um array capaz de cadastrar, em cada posição do array, cada atributo do cliente e da sua respectiva conta.
